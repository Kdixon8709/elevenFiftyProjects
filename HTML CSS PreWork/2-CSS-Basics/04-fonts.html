Font

In CSS, there are two types of font family names:

  • generic family: a group of font families with a similar look (ex: Serif)
  • font family: a specific font family (ex: "Time New Roman" or Arial)

The font-family property should hold serveral font names as a fallback in case the browser doesn't support the first font you picked.
Always start with the font you want, and end with a generic family. Also, if the font family. Also, if the family is more than one word, it must be in
quotation marks (ex: "Times New Roman")

Here is a list of the CSS font properties:

font
sets all the font properties in one declaration
font-family:
specifies the font family for text
font-size
specifies the font size of text
font-weight
specifies the weight of a font
