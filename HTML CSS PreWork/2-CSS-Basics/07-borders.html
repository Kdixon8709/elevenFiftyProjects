Borders

The border property allows you to specify the style, width, and color of and element's border.

Border Style

The border-style property can take up to four values (top border, right border, bottom, and left border). The following border-style values are allowed:

  • dotted: dotted border
  • dashed: dashed border
  • solid: solid border
  • double: double border
  • groove:3D grooved border, depends on border-color value
  • ridge: 3D ridged border, depends on border-color value
  • inset: 3D inset border, depends on the border-color value
  • none: no border
  • hidden: hidden border
  • mixed: you can set a different border style for the top, bottom, right, and left

Border Width

Very IMPORTANT: border-width will not work unless you have border-style set !!

Border-width can be set to a specific size (px, em, etc) or you can use of the following pre-set settings: thin, medium, thick. The border-width property can also take up to four values (top, right, bottom, left)

Border Color

Very IMPORTANT: border-color will not work unless you have border-style set!

Border-color can be set by the following ways

  • name: specify a color such as "blue"
  • Hex: specify a hex value, like "#eee"
  • RBG: specify a RGB value, like "rgb (100, 100, 100)
